# duedl0r/claude-plugins

Claude Code plugin marketplace by duedl0r.

## Install

```bash
claude plugin marketplace add duedl0r/claude-plugins
claude plugin install rs
```

## Available plugins

- [`rs`](plugins/rs/README.md) — structured development workflow (`/rs:grill-me`, `/rs:tdd`, `/rs:code`, `/rs:write-tech-spec`, `/rs:write-issue`, `/rs:show`)
